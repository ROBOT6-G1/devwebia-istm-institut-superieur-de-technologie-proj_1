Configuration Domaine Personnalisé & DNS

Comment faire pointer votre propre domaine (ex: ecole-avenir.mg) vers ce site ?

Connectez-vous chez votre fournisseur de nom de domaine (OVH, GoDaddy, LWS, Namecheap...) et ajoutez les enregistrements DNS suivants :

Type A (Racine @) : 76.76.21.21

Type CNAME (www) : cname.vercel-dns.com

Le certificat SSL HTTPS gratuit sera automatiquement généré par l'infrastructure Vercel de DEVWEBIA.
